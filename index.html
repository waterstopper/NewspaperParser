To get started, you need to export JSON -- How to process the day -- 1. Set the date of the day 2. (optional) remove hyphenation in the text of the day 3. Make sure that the title of the movie is in the following quotation marks: "". If you miss this, then everything will break 4. Click "Add a day" -- Remarks -- 1. When updating the page, all data is erased, so it's worth exporting the current progress from time to time (it will appear on the screen) and copying it somewhere; 2. The names of films and cinemas are register-independent (that is, "ColosSeum" and "ColosseuM" are the same thing). You only need to make sure that there are no typos in the names;
